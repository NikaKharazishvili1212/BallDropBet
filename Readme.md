# Ball Betting Mini Game

My first complete game project, built while following a game development course.
It started as a simple experiment — drop some balls, see which one reaches the bottom first —
and grew into a proper mini game with scoring, lives, randomized obstacles, and a persistent high score.

Every round is completely random. Obstacles change speed and direction, balls spawn at different
positions, and nothing plays out the same way twice.

## Gameplay & Features
- **Choose a ball** — pick one of 5 colored balls before the round starts
- **Press Start** — all balls drop from above with a random position offset
- **Watch** — balls fall through a gauntlet of randomized obstacles
- **Score** — your result depends on where your chosen ball places:
  - 1st: +1 life, +100 score
  - 2nd: +60 score
  - 3rd–5th: −1 life, lower score
- **Game over** at 0 lives — high score is saved between sessions
